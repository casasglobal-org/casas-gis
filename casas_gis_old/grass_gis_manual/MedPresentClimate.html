MedPresentClimate help
MedPresentClimate [-xlducgrpm] SaveDir=string longitude=integer latitude=integer year=integer parameter=integer interpolation=string lowercut=float uppercut=float legend1=string region=string[,string,...] alt=float resolution=integer [colorRuleDivergent=string] [colorRuleRegular=string] [--verbose] [--quiet]

Flags:

-x
Use an overall (compound) range for all maps (absolute max and min)
-l
Use also same legend bar for all maps (i.e. bar will extend to overall max and min)
-d
Use divergent, zero-centered color pattern (requires positive max and negative min)
-u
Use cutting point to mask high values
-c
Do not interpolate stations above clipping altitude
-g
Black and white output instead of color
-r
Write a report with raster statistics
-p
Produce barchart plots summarizing raster statistics
-m
Map only inside olive growing area
--verbose
Verbose module output
--quiet
Quiet module output

Parametri:

SaveDir=string
Name for directory where to store output maps
longitude=integer
Longitude (X) column number
Default: 5
latitude=integer
Latitude (Y) column number
Default: 6
year=integer
Year column number
Default: 11
parameter=integer
Parameter to map
interpolation=string
Interpolation method
Opzioni: inverse_distance_weighting,bicubic_spline
Default: inverse_distance_weighting
lowercut=float
Cutting point to mask low values
Default: 0
uppercut=float
Cutting point to mask high values (please, also check option below)
Default: 0
legend1=string
Text for legend - first line (file name of HTML summary)
region=string[,string,...]
FAO ecological regions (separated by space). Use -1 to map whole region
Default: 21 22 23 25 31 32 33 35
alt=float
Altitude (meters) above which to clip
Default: 700
resolution=integer
Resolution of output figure (single=1, double=2, quad=4) relative to computational region
Opzioni: 1,2,4
Default: 1
colorRuleDivergent=string
Select or input a DIVERGENT color pattern (hyphen-separated) using standard GRASS colors (see r.colors manual page for color names) or R:G:B triplets
Default: 32:96:255-32:159:255-32:191:255-0:207:255-42:255:255-85:255:255-127:255:255-170:255:255-255:255:84-255:240:0-255:191:0-255:168:0-255:138:0-255:112:0-255:77:0-255:0:0
colorRuleRegular=string
Select or input a REGULAR color pattern (hyphen-separated) using standard GRASS colors (see r.colors manual page for color names) or R:G:B triplets
Default: 0:0:255-0:64:255-0:128:255-0:191:255-0:255:255-0:255:255-0:255:223-0:255:159-0:255:74-0:255:0-74:255:0-159:255:0-223:255:0-255:255:0-255:255:0-255:191:0-255:128:0-255:64:0-255:0:0
