india - Map the output of CASAS models for India

PAROLE CHIAVE

models

SINOPSI

india
india help
india [-wgelxaducrp] states=string[,string,...] SaveDir=string longitude=integer latitude=integer year=integer parameter=integer interpolation=string numpoints=integer lowercut=float uppercut=float legend1=string alt=float resolution=string [colorRuleDivergent=string] [colorRuleRegular=string] [low_bar_col=float] [up_bar_col=float] [--verbose] [--quiet]

Flag:

-w
Modify extent of legend color bar (using low and high input values)
-g
Black and white output instead of color
-e
Use histogram-equalized color rule
-l
Scaling logaritmico
-x
Use an overall (compound) range for all maps (absolute max and min)
-a
Use also same legend bar for all maps (i.e. bar will extend to overall max and min)
-d
Use divergent, zero-centered color pattern (requires positive max and negative min)
-u
Use cutting point to mask high values
-c
Do not interpolate stations above clipping altitude
-r
Write a report with raster statistics (not implemented here; usually done in R)
-p
Produce barchart plots summarizing raster statistics
--verbose
Output verboso del modulo
--quiet
Output quieto del modulo

Parametri:

states=string[,string,...]
Specify multiple states using their postal code (separated by space). To map the whole country, use India (Andaman and Nicobar islands excluded)
Default: India
SaveDir=string
Name for directory where to store output maps
longitude=integer
Longitude (X) column number
Default: 5
latitude=integer
Latitude (Y) column number
Default: 6
year=integer
Year column number
Default: 11
parameter=integer
Parameter to map
interpolation=string
Interpolation method
Opzioni: idw,bspline
Default: idw
numpoints=integer
Number of interpolation points (defaulting to 4 as it is a regular grid)
Default: 4
lowercut=float
Cutting point to mask low values
Default: 0
uppercut=float
Cutting point to mask high values (please, also check option below)
Default: 0
legend1=string
Text for legend - first line (file name of HTML summary)
alt=float
Altitude (meters) above which to clip
Default: 2000
resolution=string
Resolution of output figure (single=1, double=2, quad=4)
Opzioni: 1,2,4
Default: 1
colorRuleDivergent=string
Input a divergent color pattern (hyphen-separated) using standard GRASS colors (see r.colors manual page for color names) or R:G:B triplets (input used only -x and -d flags)
colorRuleRegular=string
Input a regular color pattern (hyphen-separated) using standard GRASS colors (see r.colors manual page for color names) or R:G:B triplets (input used with -x flag)
low_bar_col=float
Lower limit for legend color bar when -w option is enabled
up_bar_col=float
Upper limit for legend color bar when -w option is enabled
